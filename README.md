## 功能实现的目标

1基于springBoot平台，集成redis，搭建环境  
2实现注册、登录、商品展示、商品详情、订单详情以及秒杀抢购功能  
3添加布隆过滤器防缓存穿透  
4添加消息队列消峰  
5利用jmeter进行压测  
6master-slave服务器，分摊DB压力
